【札记】完全非线性关卡设计

发表于2024-03-28|更新于2024-05-20|游戏设计

San Franciso关于非线性关卡设计的一些内容，内容比较精炼而且比较零散，谈到了一些非线性关卡的典型Layout，并分享了解决非线性关卡设计中典型问题的一些心得。演讲者感觉比较紧张…

B站搬运中文视频链接：

What’s Radically Nonlinear Levels

  • Nonlinear Sandox (Hitman)
  • Deeper gameplay, more choices, map knowledge(对地图的掌握、探索奖励), destroy anything an option, more realistic, replayability

Layout

  • Symmertrical Layout –低风险中心线，围绕中心线建立节奏

  • Concentric Layout – 攻击城堡

  • Node Layout – 遗迹、贫民窟，高低紧张、推进、连续叙事

  • Field Layout – 拉锯战

How to Fix Issues

  • Too linear – More space, more options
  • One solution – Trust the gameplay, give it room
  • No structure – Use layout, theme
  • Not ractive – Balance around the player
  • Feels “random”,”empty” – Communicate more, logical narration, more meaningful event
  • Not enough gameplay – Mechanis matter！
